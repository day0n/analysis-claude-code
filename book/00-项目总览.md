# Claude Code 源码分析 - 项目总览

## 项目概述

Claude Code 是 Anthropic 官方推出的 CLI 工具，允许用户通过终端与 Claude AI 进行交互式对话，执行软件工程任务。项目使用 TypeScript 编写，基于 React Ink 构建终端 UI，采用 Bun 作为运行时和打包工具。

## 技术栈

| 技术 | 用途 |
|------|------|
| TypeScript | 主要开发语言 |
| React + Ink | 终端 UI 框架 |
| Bun | 运行时 & 打包工具 |
| Zod | 运行时类型校验 |
| Yoga Layout | Flexbox 布局引擎（纯 TS 移植） |
| OpenTelemetry | 可观测性/遥测 |
| OAuth 2.0 | 认证体系 |
| WebSocket / SSE | 实时通信 |
| MCP (Model Context Protocol) | 工具扩展协议 |

## 源码统计

- **总文件数**: 1903 个 TypeScript/TSX 文件
- **模块目录数**: 35 个顶层模块
- **根级文件数**: 18 个核心入口文件

## 模块架构图

```
src/
├── main.tsx                    # 应用主入口
├── Tool.ts                     # 工具抽象基类
├── Task.ts                     # 任务类型定义
├── QueryEngine.ts              # 查询引擎
│
├── entrypoints/                # 入口点（CLI/SDK/MCP）
├── bootstrap/                  # 应用启动 & 全局状态
├── cli/                        # CLI 传输层 & 处理器
├── commands/                   # 斜杠命令注册
├── screens/                    # 主要 UI 屏幕（REPL/Doctor/Resume）
├── components/                 # React UI 组件库
├── ink/                        # Ink 终端渲染引擎（深度定制）
│
├── query/                      # 查询执行核心
├── services/                   # 服务层（API/分析/压缩）
├── tools/                      # AI 工具实现（Bash/Read/Write/Edit...）
├── skills/                     # 技能系统（/commit /review 等）
├── plugins/                    # 插件系统
├── hooks/                      # 钩子系统（生命周期事件）
│
├── state/                      # 应用状态管理（Zustand-like）
├── context/                    # 上下文收集（git/CLAUDE.md）
├── constants/                  # 常量定义
├── types/                      # 全局类型定义
├── schemas/                    # Zod Schema 定义
│
├── bridge/                     # Web/IDE 桥接通信
├── coordinator/                # 多会话协调器
├── remote/                     # 远程会话管理
├── server/                     # HTTP/WebSocket 服务器
│
├── keybindings/                # 快捷键系统
├── vim/                        # Vim 模式支持
├── voice/                      # 语音模式
├── buddy/                      # 伴侣系统（彩蛋）
│
├── utils/                      # 工具函数库（500+ 文件）
├── native-ts/                  # 纯 TS 原生实现（文件索引/Yoga/颜色diff）
├── memdir/                     # 自动记忆系统
├── migrations/                 # 配置迁移
├── moreright/                  # 外部构建存根
├── outputStyles/               # 输出样式加载
├── upstreamproxy/              # 上游代理
└── tasks/                      # 后台任务管理
```

## 核心数据流

```
用户输入 → PromptInput 组件
    → REPL.tsx (主屏幕)
        → query() 函数 (query/目录)
            → callModel() (services/api/)
                → Claude API (流式响应)
            → 工具调用处理 (tools/)
            → 权限检查 (hooks/)
            → 停止钩子 (query/stopHooks.ts)
     (components/ + ink/)
    → 会话持久化 (utils/sessionStorage)
```
